COMPLETE INSTRUCTIONS: Build the SEO & Marketing Page
WHAT YOU ARE BUILDING:
 You are creating an SEO & Marketing page for YoForex forum admin panel. This page helps admins manage SEO meta tags for better search engine rankings, create marketing campaigns, and analyze SEO performance to grow organic traffic to the forum.

PART 1: BACKEND (Server Side - SEO & Marketing Data)
What the Backend Does
The backend manages meta tags for all forum pages, tracks SEO analytics, stores marketing campaign data, and monitors search engine performance.
Create These Files:
File 1: SEO & Marketing Routes (routes/seo.js)
Create URLs for SEO operations:
/api/seo/meta-tags - Get all meta tags


/api/seo/meta-tags/:id - Get single page meta tags


/api/seo/meta-tags/create - Create meta tags for new page


/api/seo/meta-tags/:id/update - Update meta tags


/api/seo/meta-tags/:id/delete - Delete meta tags


/api/seo/meta-tags/bulk-update - Update multiple pages


/api/seo/campaigns - Get all marketing campaigns


/api/seo/campaigns/create - Create new campaign


/api/seo/campaigns/:id/update - Update campaign


/api/seo/campaigns/:id/delete - Delete campaign


/api/seo/analytics/traffic - Get organic traffic data


/api/seo/analytics/keywords - Get top keywords


/api/seo/analytics/pages - Get top performing pages


/api/seo/analytics/search-console - Get Google Search Console data


/api/seo/sitemap/generate - Generate XML sitemap


/api/seo/robots/update - Update robots.txt


File 2: SEO Service (services/seoService.js)
Functions that handle SEO operations:
Function 1: Get All Meta Tags
Query seo_meta_tags table


For each page/content include:


Page ID


Page title (e.g., "YoForex - Forex Trading Forum")


Page type (homepage, category, thread, user profile, etc.)


Page URL


Meta title (SEO title, 50-60 characters)


Meta description (SEO description, 150-160 characters)


Meta keywords (comma-separated)


Open Graph title (for social media)


Open Graph description


Open Graph image URL


Twitter card type


Canonical URL


Created date


Last updated


Return all pages with meta tags


Function 2: Create Meta Tags
Insert new entry in seo_meta_tags table


Required fields: page_type, page_url, meta_title, meta_description


Optional: keywords, og tags, twitter tags


Validate:


Meta title 50-60 characters


Meta description 150-160 characters


URL format valid


Set created_by to admin username


Return new meta tag entry


Function 3: Update Meta Tags
Find meta tags by ID


Update: meta_title, meta_description, keywords, og tags


Validate character limits


Set updated_at timestamp


Record change in admin_actions


Return updated data


Function 4: Get Marketing Campaigns
Query marketing_campaigns table


For each campaign include:


Campaign ID


Campaign name (e.g., "Facebook Ads - Gold Trading")


Campaign type (social media, email, paid ads, content)


Platform (Facebook, Google, Twitter, Email, etc.)


Campaign URL with UTM parameters


Start date and end date


Budget allocated


Clicks received


Conversions (sign-ups, purchases)


Cost per click (CPC)


Return on investment (ROI)


Status (active, paused, completed)


Sort by date (newest first)


Filter by: type, platform, status


Function 5: Create Marketing Campaign
Insert campaign in marketing_campaigns table


Required: name, type, platform, start_date


Generate UTM parameters automatically:


utm_source (facebook, google, email, etc.)


utm_medium (social, cpc, email, etc.)


utm_campaign (campaign name slugified)


Create tracking URL


Set created_by to admin


Return campaign with tracking URL


Function 6: Get SEO Analytics
Query analytics data or integrate with Google Analytics API


Calculate:


Total organic traffic (last 30 days)


Organic traffic trend (daily breakdown)


Top 10 keywords bringing traffic


Top 10 pages by organic views


Average position in search results


Click-through rate (CTR)


Bounce rate for organic traffic


New users from organic search


Compare to previous period


Function 7: Get Search Console Data
Integrate with Google Search Console API


Fetch:


Total impressions in search results


Total clicks from search


Average CTR


Average position


Top queries (keywords users searched)


Top pages


Countries sending traffic


Devices (mobile, desktop, tablet)


Return data for date range


Function 8: Generate XML Sitemap
Query all public pages:


Homepage


Category pages


Thread pages


User profiles (public)


Static pages


For each page include:


URL


Last modified date


Change frequency (daily, weekly, monthly)


Priority (0.0 to 1.0)


Generate XML format sitemap


Save to /sitemap.xml


Submit to search engines


Return sitemap URL


Function 9: Update Robots.txt
Read current robots.txt file


Allow admin to edit rules:


User-agent specifications


Disallow rules (blocked URLs)


Allow rules


Sitemap location


Crawl-delay settings


Validate syntax


Save updated file


Return success
